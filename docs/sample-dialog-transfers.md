# Пример тестового диалога — переводы

Используйте этот сценарий для синтеза тестового аудиофайла через TTS.

**Тема:** переводы  
**Длительность:** ~3 минуты  
**Участники:** Оператор (Анна), Клиент (Елена)

---

**Оператор:** Добрый день, МТБанк, меня зовут Анна, чем могу помочь?

**Клиент:** Здравствуйте. Мне нужно перевести деньги на счёт в другом банке.

**Оператор:** Конечно, подскажите, пожалуйста, это перевод внутри Беларуси или за границу?

**Клиент:** Внутри страны, на счёт моей сестры в другом банке.

**Оператор:** Хорошо. Перевод по номеру счёта доступен в приложении и в интернет-банке. Какую сумму планируете отправить?

**Клиент:** Пятьсот рублей.

**Оператор:** Для перевода до тысячи рублей в сутки комиссия не взимается, если переводите с карты МТБанка. Средства обычно поступают в течение одного рабочего дня.

**Клиент:** А если мне нужно отправить срочно, в тот же день?

**Оператор:** Тогда выберите в приложении опцию срочного перевода по системе мгновенных платежей. Комиссия составит один рубль пятьдесят, зачисление — в течение нескольких минут.

**Клиент:** Понятно. Мне понадобятся реквизиты получателя?

**Оператор:** Да, номер счёта получателя, код банка и ФИО. Если переводите по номеру телефона через систему мгновенных платежей, достаточно номера и выбранного банка получателя.

**Клиент:** У меня есть только номер счёта и название банка.

**Оператор:** Тогда оформляйте обычный перевод по реквизитам. Код банка я могу подсказать по названию — как называется банк получателя?

**Клиент:** Беларусбанк.

**Оператор:** Записала. В приложении выберите «Переводы», затем «По реквизитам», введите счёт и сумму. Проверьте ФИО получателя перед подтверждением.

**Клиент:** А лимит на переводы есть? Я иногда отправляю больше.

**Оператор:** Базовый лимит — три тысячи рублей в сутки через приложение. Лимит можно увеличить в настройках безопасности или в отделении при предъявлении документа.

**Клиент:** Хорошо, попробую через приложение.

**Оператор:** Отлично. Если перевод не пройдёт, сохраните номер операции и позвоните нам — проверим статус. Могу отправить краткую инструкцию на email?

**Клиент:** Да, пожалуйста.

**Оператор:** Подскажите ваш email.

**Клиент:** Елена-собака-пример-точка-бай.

**Оператор:** Записала, письмо придёт в течение нескольких минут. Есть ещё вопросы?

**Клиент:** Нет, спасибо, всё ясно.

**Оператор:** Спасибо за обращение в МТБанк, хорошего дня!

**Клиент:** И вам, до свидания.

---

## Подсказки для синтеза

```bash
pip install edge-tts pydub

# Оператор — женский голос
edge-tts --voice ru-RU-SvetlanaNeural --text "Добрый день, МТБанк..." --write-media operator.mp3

# Клиент — женский голос
edge-tts --voice ru-RU-DariyaNeural --text "Здравствуйте..." --write-media client.mp3
```

Склейте реплики с **паузой 0,7 с между репликами** через `pydub` или `ffmpeg`:

```python
from pydub import AudioSegment

PAUSE_MS = 700  # 0.7 сек между репликами
silence = AudioSegment.silent(duration=PAUSE_MS)

# Чередуйте operator.mp3, silence, client.mp3, silence, ...
# Экспорт: combined.export("test_data/dialog-transfers.mp3", format="mp3")
```

Телефонное качество (8 kHz):

```bash
ffmpeg -i dialog-transfers.mp3 -ar 8000 -ac 1 dialog-transfers-tel.mp3
```

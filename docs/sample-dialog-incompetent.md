# Пример тестового диалога — некомпетентный оператор

Используйте этот сценарий для синтеза короткого тестового аудиофайла через TTS.

**Назначение:** проверка, что LLM не засчитывает `solution_provided`, если оператор не дал конкретного решения или следующего шага.  
**Тема:** переводы (проблема с зачислением)  
**Длительность:** ~1 минута  
**Участники:** Оператор (Анна), Клиент (Игорь)

**Ожидаемая оценка качества:** greeting ✓, need_detection ✓ (частично), **solution_provided ✗**, farewell ✓.

---

**Оператор:** Добрый день, МТБанк, меня зовут Анна, чем могу помочь?

**Клиент:** Здравствуйте. Вчера отправил перевод на триста рублей, деньги до сих пор не дошли получателю.

**Оператор:** А, переводы... Ну бывает, подождите ещё немного, должно дойти.

**Клиент:** Сколько ждать? Получатель уже звонит, ему нужны деньги сегодня.

**Оператор:** Я точно не знаю. Может, у них там что-то на стороне получателя.

**Клиент:** А вы можете проверить статус моей операции?

**Оператор:** У меня сейчас ничего не показывает. Попробуйте позвонить завтра, может тогда увидим.

**Клиент:** Завтра? Мне нужно решить сегодня. Что мне делать?

**Оператор:** Ну, тогда не знаю, чем помочь. Извините.

**Клиент:** Хорошо, до свидания.

**Оператор:** До свидания.

---

## Подсказки для синтеза

```bash
pip install edge-tts pydub

# Оператор — женский голос
edge-tts --voice ru-RU-SvetlanaNeural --text "Добрый день, МТБанк..." --write-media operator.mp3

# Клиент — мужской голос
edge-tts --voice ru-RU-DmitryNeural --text "Здравствуйте..." --write-media client.mp3
```

Склейте реплики с **паузой 0,7 с между репликами** через `pydub` или `ffmpeg`:

```python
from pydub import AudioSegment

PAUSE_MS = 700  # 0.7 сек между репликами
silence = AudioSegment.silent(duration=PAUSE_MS)

# Чередуйте operator.mp3, silence, client.mp3, silence, ...
# Экспорт: combined.export("test_data/dialog-incompetent.mp3", format="mp3")
```

Телефонное качество (8 kHz):

```bash
ffmpeg -i dialog-incompetent.mp3 -ar 8000 -ac 1 dialog-incompetent-tel.mp3
```

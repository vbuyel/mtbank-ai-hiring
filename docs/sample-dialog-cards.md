# Пример тестового диалога — карты

Используйте этот сценарий для синтеза тестового аудиофайла через TTS.

**Тема:** карты  
**Длительность:** ~3 минуты  
**Участники:** Оператор (Анна), Клиент (Ольга)

---

**Оператор:** Добрый день, МТБанк, меня зовут Анна, чем могу помочь?

**Клиент:** Здравствуйте. Хочу узнать про вашу дебетовую карту с кэшбэком.

**Оператор:** Конечно. У нас есть карта «МТБанк Кэшбэк» — возврат до трёх процентов в выбранных категориях и один процент на остальные покупки. Вас интересует оформление новой карты или условия по действующей?

**Клиент:** Новой. Я пока не клиент банка.

**Оператор:** Тогда можно оформить онлайн: заявка занимает около десяти минут, карту доставят курьером или можно забрать в отделении. Обслуживание первый год бесплатное.

**Клиент:** А какие документы нужны?

**Оператор:** Паспорт или другой документ, удостоверяющий личность. Для резидентов дополнительных справок не требуется. При доставке курьер проверит документ и выдаст карту.

**Клиент:** Кэшбэк как начисляется — сразу или в конце месяца?

**Оператор:** Баллы начисляются на следующий день после покупки, использовать можно в приложении — обменять на рубли или оплатить ими связь и другие услуги партнёров.

**Клиент:** Можно выбрать категории с повышенным кэшбэком?

**Оператор:** Да, каждый месяц в приложении выбираете две категории — например, продукты и аптеки — и получаете три процента по ним.

**Клиент:** А снятие наличных в банкоматах других банков?

**Оператор:** В банкоматах МТБанка снятие бесплатное. В чужих банкоматах — по тарифу: первые два снятия в месяц без комиссии, далее один процент, минимум два рубля.

**Клиент:** Звучит неплохо. Можно оформить сразу с мобильным приложением?

**Оператор:** Да, после одобрения заявки вы получите доступ к интернет-банку и приложению, карту можно добавить в Apple Pay или Google Pay до получения пластика.

**Клиент:** Хорошо, оформлю онлайн. Есть лимит на бесплатное обслуживание после первого года?

**Оператор:** Со второго года обслуживание бесплатно при тратах от ста рублей в месяц или при среднем остатке от пятисот рублей. Иначе — три рубля в месяц.

**Клиент:** Понятно. Можете прислать ссылку на заявку?

**Оператор:** Конечно, отправлю на email или в SMS. Что удобнее?

**Клиент:** На email, пожалуйста.

**Оператор:** Подскажите ваш email.

**Клиент:** Ольга-собака-пример-точка-бай.

**Оператор:** Записала, письмо с ссылкой и условиями придёт в течение нескольких минут. Есть ещё вопросы?

**Клиент:** Нет, всё понятно, спасибо.

**Оператор:** Спасибо за интерес к МТБанку, хорошего дня!

**Клиент:** И вам, до свидания.

## Подсказки для синтеза

```bash
pip install edge-tts pydub

# Оператор — женский голос
edge-tts --voice ru-RU-SvetlanaNeural --text "Добрый день, МТБанк..." --write-media operator.mp3

# Клиент — женский голос
edge-tts --voice ru-RU-DariyaNeural --text "Здравствуйте..." --write-media client.mp3
```

Склейте реплики с **паузой 0,7 с между репликами** через `pydub` или `ffmpeg`:

```python
from pydub import AudioSegment

PAUSE_MS = 700  # 0.7 сек между репликами
silence = AudioSegment.silent(duration=PAUSE_MS)

# Чередуйте operator.mp3, silence, client.mp3, silence, ...
# Экспорт: combined.export("test_data/dialog-cards.mp3", format="mp3")
```

Телефонное качество (8 kHz):

```bash
ffmpeg -i dialog-cards.mp3 -ar 8000 -ac 1 dialog-cards-tel.mp3
```

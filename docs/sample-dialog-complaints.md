# Пример тестового диалога — жалобы

Используйте этот сценарий для синтеза тестового аудиофайла через TTS.

**Тема:** жалобы  
**Длительность:** ~3 минуты  
**Участники:** Оператор (Анна), Клиент (Сергей)

---

**Оператор:** Добрый день, МТБанк, меня зовут Анна, чем могу помочь?

**Клиент:** Здравствуйте. Я хочу пожаловаться — с моей карты списали деньги, которых я не совершал.

**Оператор:** Понимаю, это неприятная ситуация. Давайте разберёмся. Подскажите, пожалуйста, когда было списание и на какую сумму?

**Клиент:** Вчера вечером, сто двадцать рублей. Магазин электроники, я там ничего не покупал.

**Оператор:** Спасибо. Для начала нужно заблокировать карту, чтобы предотвратить новые списания. Вы согласны?

**Клиент:** Да, блокируйте сразу.

**Оператор:** Хорошо, карта будет заблокирована в течение нескольких минут. Подскажите последние четыре цифры карты и ваше ФИО для идентификации.

**Клиент:** Карта заканчивается на четыре пять шесть семь, Сергей Ковалёв.

**Оператор:** Спасибо, Сергей. Вижу операцию на сто двадцать рублей от вчера, двадцать три часа сорок минут. Вы карту никому не передавали и данные в интернете не вводили?

**Клиент:** Нет, карта у меня, никуда данные не вводил.

**Оператор:** Тогда оформим заявление на оспаривание операции. Срок рассмотрения — до тридцати календарных дней, обычно быстрее. Деньги могут быть временно возвращены на счёт после проверки.

**Клиент:** А сколько ждать? Мне эти деньги нужны на оплату.

**Оператор:** Предварительное решение часто принимают за пять рабочих дней. Я зафиксирую обращение как срочное и укажу, что карта у клиента, это ускорит проверку.

**Клиент:** Хорошо. Новую карту как получить?

**Оператор:** Перевыпуск бесплатный, карту можно заказать в приложении или забрать в отделении через три рабочих дня. Доставка курьером — за дополнительную плату по тарифам банка.

**Клиент:** Закажу в отделении, мне так удобнее.

**Оператор:** Записала. Номер вашего обращения — я продиктую и отправлю в SMS. Также пришлю на email памятку, какие документы могут понадобиться.

**Клиент:** Да, отправьте, пожалуйста.

**Оператор:** Подскажите email или подтвердите номер телефона для SMS.

**Клиент:** Сергей-точка-ковалев-собака-пример-точка-бай, телефон тот же, с которого звоню.

**Оператор:** Отлично. В течение десяти минут получите номер обращения и инструкцию. Есть ещё вопросы?

**Клиент:** Нет, главное — вернуть деньги и заблокировать карту.

**Оператор:** Карта уже в процессе блокировки, заявление оформлено. Спасибо за обращение, мы свяжемся с вами по результатам проверки.

**Клиент:** Спасибо, до свидания.

**Оператор:** Хорошего дня, Сергей!

---

## Подсказки для синтеза

```bash
pip install edge-tts pydub

# Оператор — женский голос
edge-tts --voice ru-RU-SvetlanaNeural --text "Добрый день, МТБанк..." --write-media operator.mp3

# Клиент — мужской голос
edge-tts --voice ru-RU-DmitryNeural --text "Здравствуйте..." --write-media client.mp3
```

Склейте реплики с **паузой 0,7 с между репликами** через `pydub` или `ffmpeg`:

```python
from pydub import AudioSegment

PAUSE_MS = 700  # 0.7 сек между репликами
silence = AudioSegment.silent(duration=PAUSE_MS)

# Чередуйте operator.mp3, silence, client.mp3, silence, ...
# Экспорт: combined.export("test_data/dialog-complaints.mp3", format="mp3")
```

Телефонное качество (8 kHz):

```bash
ffmpeg -i dialog-complaints.mp3 -ar 8000 -ac 1 dialog-complaints-tel.mp3
```
